# Product Review GSAP

## Summary
This is a modern perfume product showcase built with React, Vite, GSAP, and Three.js.
It includes scroll-driven storytelling sections and a 3D bottle experience.

## Run Locally
1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

4. Preview production build:

```bash
npm run preview
```
